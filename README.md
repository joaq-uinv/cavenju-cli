# C A V E N J U CLI

To install this package open a terminal and run

`npm i -g cavenju-cli`

To run it,

`canveju`

Alternatively, you can install it by running the following commands:

- `npm i -g typescript`

- `git clone https://github.com/joaq-uinv/cavenju-cli.git && npm i`

To run it,

`npm run build && npm start`
